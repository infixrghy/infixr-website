# SPEC

## §G GOAL
static infixr.com rebuild. html+css. minimal js. mobile-OK. GH Pages host.

## §C CONSTRAINTS
- static HTML + CSS only. ⊥ framework (Tailwind, Bootstrap, etc).
- JS budget ≤ 5 KB total across all `src/js/*.js`.
- new JS ! Ari approval first. ⊥ silent add.
- modern CSS OK: `@layer`, `color-mix(in oklab)`, `clamp()`, container queries, `:user-invalid`, `100dvh`, `backdrop-filter`, `scroll-snap`, `animation-timeline: view()`, `aspect-ratio`, `@supports`, `@property`, `@starting-style` + `transition-behavior: allow-discrete`, `:has()`, `field-sizing: content`, `text-wrap: balance|pretty`. newer feats (`@property`/`@starting-style`/`field-sizing`) progressive-enhance via `@supports` where engine coverage is partial (V40–V44).
- CSS ! native nesting (nest child rules in parent w/ `&`). ⊥ flat repeated selector chains. ⊥ preprocessor (Sass/Less) — browser-native only.
- mobile-first. ! hold at 360 px width.
- progressive enhance. unsupported features → graceful fallback, never blank/broken.
- Bun toolchain. ⊥ npm/yarn.
- layout: `src/` = hand-edited source of truth; `build.ts` emits `public/` (generated, gitignored).
- GH Pages served from `public/` via GitHub Actions (`.github/workflows/deploy.yml`). `.nojekyll` emitted into `public/`.
- build pipeline = Effect.ts (Schema + Option, ⊥ null) as build-time-only LIBRARY (ships 0 client bytes — ✓ ⊥-framework). templating = typed `Data → string` render fns, ⊥ templating DSL. markdown via `marked`, build-time only, 0 client bytes.
